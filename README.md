# CS50 Mario

This project was completed as part of Harvard CS50x.

## Description
A C program that prints a double half-pyramid of a specified height using hashes (#), inspired by Mario games.

The user enters a height between 1 and 8, and the program generates the pyramid using loops and conditionals.

## Technologies Used
- C
- CS50 Library

## Concepts Learned
- Loops
- Nested loops
- User input validation
- Problem Solving
